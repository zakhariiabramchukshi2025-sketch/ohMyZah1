OH MY ZAH 



Welcome to OH MY ZAH

This project is a multifunctional c++ based console program for ZSH (+bash and powerShell in future releases) which provides you with:
- basic (zsh) console command line framework
- Remote access to devices you connect
- basic encryption functions
- image making tool
- ... (more in future releases)
